Universal Theme Library for Vanilla JavaScript

Welcome to the Omni-Themes vanilla JavaScript demo! This demonstrates how to use our universal theme library in any JavaScript application without framework dependencies.

Installation

npm install @madooei/omni-themes

Basic Usage

import { createThemeStore } from '@madooei/omni-themes';

// Create theme store
const {
  themes,           // Available themes array
  $theme,           // Current theme atom
  $resolvedTheme,   // Resolved theme atom (handles 'system')
  $systemTheme,     // System preference atom
  setTheme,         // Function to change theme
  applyThemeScriptString  // Script for FOUC prevention
} = createThemeStore({
  themes: ['light', 'dark', 'blue', 'green'],
  enableSystem: true,
  enableColorScheme: true
});

// Listen for theme changes
$theme.listen((theme) => {
  console.log('Theme changed to:', theme);
});

// Change theme
setTheme('dark');

FOUC Prevention

// Inject theme script in HTML head to prevent flash
const themeScript = document.createElement('script');
themeScript.innerHTML = applyThemeScriptString;
document.head.appendChild(themeScript);
