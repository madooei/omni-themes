How to Test FOUC:

  1. With FOUC Prevention (Default): The page loads with the correct theme immediately - no flash
  2. Without FOUC Prevention: Click "Toggle FOUC Prevention" then "Reload Page" - you'll see a bright pink/green flash before the correct theme loads
  3. Test Theme Switching: Use the theme buttons to see instant changes without page reload
  4. Test Persistence: Set a theme, reload the page, and see if it persists correctly

What's Happening:

  • Default styles (pink/green): These are the fallback styles that show before JavaScript loads
  • FOUC Prevention: Injects a script in the <head> that applies the correct theme before the page renders
  • Without Prevention: The page shows default styles until JavaScript loads and applies the theme
